# Homepage Image Guide

## Where to Place Images on the Main Page

### 1. **Hero Section Background** (Optional)
- **Location:** Main hero section behind the text
- **Folder:** Create `images/homepage/hero-background.jpg`
- **Usage:** Large background image for the main hero section
- **Recommended size:** 1920x1080 pixels (high resolution for full screen)

### 2. **About Section Image** 
- **Location:** Right side of the "Inspiring Spaces" section
- **Folder:** Create `images/homepage/about-image.jpg`
- **Usage:** Shows your design work or studio photo
- **Recommended size:** 800x600 pixels

### 3. **Project Preview Images** (Portfolio Section)
These are the images that show on the main page before clicking into galleries:

- **Wellness Retreat Preview:** `images/homepage/wellness-preview.jpg`
- **Minimalist Residence Preview:** `images/homepage/minimalist-preview.jpg` 
- **Spa Wellness Preview:** `images/homepage/spa-preview.jpg`
- **Hotel Penthouse Preview:** `images/homepage/hotel-preview.jpg`

**Recommended size:** 800x600 pixels each

### 4. **Logo** (Optional)
- **Location:** Top navigation bar
- **Folder:** `images/homepage/logo.png` or `logo.svg`
- **Usage:** Replace "Wisan Studio" text with your logo
- **Recommended size:** 200x80 pixels (PNG with transparent background)

## Folder Structure for Homepage Images

```
images/
├── homepage/
│   ├── hero-background.jpg        (Optional - hero background)
│   ├── about-image.jpg            (About section image)
│   ├── wellness-preview.jpg       (Wellness project preview)
│   ├── minimalist-preview.jpg     (Minimalist project preview)
│   ├── spa-preview.jpg            (Spa project preview)
│   ├── hotel-preview.jpg          (Hotel project preview)
│   └── logo.png                   (Optional - logo)
├── wellness-retreat/              (Gallery images)
├── minimalist-residence/          (Gallery images)
├── spa-wellness/                  (Gallery images)
└── hotel-penthouse/               (Gallery images)
```

## Priority Order

**Most Important (do these first):**
1. **Project preview images** - These show on the portfolio section
2. **About section image** - Shows your work or studio

**Optional (nice to have):**
3. **Hero background image** - Can make the hero section more visual
4. **Logo** - If you want to replace the "Wisan Studio" text

## What Images Work Best

### Project Preview Images
- Hero shots of each project
- Best/most impressive photo from each project
- Should represent the style of that project

### About Image
- Photo of your design work
- Studio photo
- Behind-the-scenes of your design process
- Professional headshot

### Hero Background
- Stunning interior design photo
- Should be subtle enough that white text shows clearly
- Architectural photography works well
