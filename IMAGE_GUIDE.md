# Photo Organization Guide

## Folder Structure

Your photos should be organized in the following folders:

```
images/
├── wellness-retreat/
│   ├── image1.jpg
│   ├── image2.jpg
│   ├── image3.jpg
│   ├── image4.jpg
│   └── image5.jpg
├── minimalist-residence/
│   ├── image1.jpg
│   ├── image2.jpg
│   ├── image3.jpg
│   ├── image4.jpg
│   └── image5.jpg
├── spa-wellness/
│   ├── image1.jpg
│   ├── image2.jpg
│   ├── image3.jpg
│   ├── image4.jpg
│   └── image5.jpg
└── hotel-penthouse/
    ├── image1.jpg
    ├── image2.jpg
    ├── image3.jpg
    ├── image4.jpg
    └── image5.jpg
```

## How to Add Your Photos

### 1. **Hillside Luxury Wellness Retreat**
Put your wellness retreat photos in: `images/wellness-retreat/`
- Name them: `image1.jpg`, `image2.jpg`, `image3.jpg`, etc.
- These will show up when users click on the "Hillside Luxury Wellness Retreat" project

### 2. **Modern Minimalist Residence**
Put your minimalist home photos in: `images/minimalist-residence/`
- Name them: `image1.jpg`, `image2.jpg`, `image3.jpg`, etc.
- These will show up when users click on the "Modern Minimalist Residence" project

### 3. **Urban Spa & Wellness Center**
Put your spa photos in: `images/spa-wellness/`
- Name them: `image1.jpg`, `image2.jpg`, `image3.jpg`, etc.
- These will show up when users click on the "Urban Spa & Wellness Center" project

### 4. **Boutique Hotel Penthouse**
Put your hotel photos in: `images/hotel-penthouse/`
- Name them: `image1.jpg`, `image2.jpg`, `image3.jpg`, etc.
- These will show up when users click on the "Boutique Hotel Penthouse" project

## Important Notes

- **File naming:** Use exactly `image1.jpg`, `image2.jpg`, etc. (lowercase, no spaces)
- **File format:** Use `.jpg`, `.jpeg`, or `.png` files
- **Image size:** Recommended size is 1200x800 pixels or similar high-resolution
- **Number of images:** You can have 3-10 images per project. If you have more or fewer than 5 images per project, let me know and I'll update the JavaScript code

## Example

If you have 3 photos for the wellness retreat:
1. Put them in `images/wellness-retreat/`
2. Name them:
   - `image1.jpg`
   - `image2.jpg` 
   - `image3.jpg`
3. Let me know you have 3 images instead of 5, and I'll update the code

## Adding New Projects

If you want to add more projects:
1. Create a new folder in `images/` (e.g., `images/new-project/`)
2. Add your photos named `image1.jpg`, `image2.jpg`, etc.
3. Let me know and I'll add the new project to the website

## Need Help?

If you need to:

- Change project names
- Add more projects
- Use different file names

Just let me know and I'll update the code accordingly!
